SalesScript

Description:

A modern web application that generates personalized sales scripts using AI. This tool empowers sales professionals to craft effective outreach by leveraging intelligent language models tailored to specific buyer personas and product offerings. It also includes a chatbot that role-plays the buyer objecting to certain things in order to aid the rep practice handling possible objections before getting on the call. Vibe-coded at a hackathon.

Technologies:

- Typescript

- React

- Ollama

- Llama3

- Material-UI
